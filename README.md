# 📚 English Janala – Vocabulary Learning Web App

English Janala is a responsive web application designed to help users learn English vocabulary easily through interactive lessons.  
The project focuses on simplicity, user-friendly design, and responsive layout for different devices.

This project was built using **HTML5, Tailwind CSS, DaisyUI, and Vanilla JavaScript**.

---

## 🚀 Features

- 🔤 Learn English vocabulary easily
- 📖 Interactive vocabulary lessons
- 🔍 Word search functionality
- 📱 Fully responsive design
- 🎨 Clean and modern user interface
- ⚡ Fast performance using Vanilla JavaScript

---

## 🛠️ Technologies Used

- **HTML5** – Structure of the website  
- **Tailwind CSS** – Styling and layout  
- **DaisyUI** – Pre-built UI components  
- **JavaScript (Vanilla JS)** – Functionality and DOM manipulation
